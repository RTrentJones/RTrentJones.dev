I'm Trent Jones — an engineer who likes tools that prove they work.

This site, and the projects linked from it, all run on **Greenlight**: a deploy + verification
harness I built. Every change ships through one loop — build → verify → promote — where the same
checks run in CI and locally, so things go live with objective confidence instead of vibes.

[Greenlight is on GitHub](https://github.com/RTrentJones/greenlight) (and npm). This blog is itself
a Greenlight consumer: an Astro site on Cloudflare Workers, gated by the same verify harness as
everything else.
